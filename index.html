Transform your mathematical formulation into a Python Finite Element simulation.

Readability

A Nutils script focuses on work flow and maths, unobscured by finite element infrastructure.

Versatility

The Nutils library provides tools; they do not enforce a strict work flow. Missing components can be added locally without losing interoperability.

Compatibility

Exposed objects are of native Python type or allow for easy conversion to leverage third party tools such as Numpy, Scipy and Matplotlib.

Performance

A Nutils code is self-optimizing and supports parallel computation. Typical scripting inefficiencies are discouraged by design.
