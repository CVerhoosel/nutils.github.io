--- layout: post title: Pip install nutils tags: news author: Gertjan van Zwieten ---

Today the Nutils project is recognized as a Python package. This means you can now install nutils releases via the pip installation tools as pip install nutils, along with all its dependencies. Though thus far only tested on linux, pip supports all major platforms and we aim to make installation of nutils a painless experience. Note that for active development the recommended mode of operation remains to work from an up to date git tree.
